# Benjamins Bestiary
Juniper Rodriguez (juniperr)

## Description
Benjamin's Bestiary is a children's 'Choose Your Own Adventure' storybook iOS app for iPad that follows Benjamin, a Beastkeeper in the kingdom of Andolonia, as he brings a mysterious creature along on his travels.

## One More Thing
The Bestiary will be a view within the app that displays the creatures the user has created after reading the story, storing the creature on the device using Core Data. On this page, users can save the image of the creature to their device and modify its name.
